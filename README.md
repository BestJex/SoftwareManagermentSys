# backendManagermentSys

自用后端软件管理系统
主要用于管理公司软件，目前通用性一般，需要使用请酌情考虑。

# web

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
